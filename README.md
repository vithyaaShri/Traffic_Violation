# Traffic Violations Insight System

## Project Overview

The Traffic Violations Insight System is a data analytics dashboard developed using Python, MySQL, and Streamlit to analyze over 2 million traffic violation records.

## Technology Stack

- Python
- Pandas
- NumPy
- MySQL
- SQLAlchemy
- Streamlit
- Plotly

## Features

- Data Cleaning
- Data Validation
- SQL Analytics
- Interactive Dashboard
- Geographic Analysis
- Vehicle Analysis
- Demographic Analysis

## Dataset Information

- Total Records: 2,067,761
- Total Columns: 43

## Run Project

```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```
